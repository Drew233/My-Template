    Install

    Use

    The use of mkdocs,I don't know enough now,so I just tell you how to use it with giuhub pages.

    Add a new repo on your github

    The first step is easy that I believe you can finish it without any doubt.And then you are advised to clone all the repo files to you computer with "git clone".

    Import project

    Transfer all the files in the project you generated with Mkdocs to the folder that you clone from github.

    The final step

    Run mkdocs gh-deploy,and mkdocs will automatically generate web pages and upload them to github.That is the most part of Mkdocs I like.
